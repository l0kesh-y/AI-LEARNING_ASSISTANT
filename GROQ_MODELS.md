# Groq Supported Models (Updated December 2025)

## Currently Used Model
- **llama-3.1-8b-instant** - Fast, efficient model for general tasks

## Alternative Models Available
- **llama-3.1-70b-versatile** - More powerful but slower
- **llama-3.2-1b-preview** - Lightweight model
- **llama-3.2-3b-preview** - Balanced performance
- **mixtral-8x7b-32768** - Good for complex reasoning
- **gemma-7b-it** - Google's Gemma model

## Model Selection Guidelines
- **Chat/Conversation**: llama-3.1-8b-instant (current choice)
- **Complex Analysis**: llama-3.1-70b-versatile
- **Fast Responses**: llama-3.2-1b-preview
- **Balanced Tasks**: llama-3.2-3b-preview
